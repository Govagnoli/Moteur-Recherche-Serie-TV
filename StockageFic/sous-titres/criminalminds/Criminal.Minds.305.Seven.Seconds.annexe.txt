Entretien cognitif (Cognitive interview)
Procédure d’interrogatoire pour le recueil des témoignages. La première version de l’entretien cognitif est basée sur les techniques suivantes :
-  réinstaurer mentalement le contexte de l’événement ;
-  se souvenir de chaque détail, même s’il apparaît au témoin comme étant sans importance, trivial ;
-  se souvenir de l’événement selon différentes directions temporelles ;
-  décrire l’événement en fonction de différents points de vue.
Une version révisée ajoute notamment des techniques de communication sociale que doit utiliser l’interviewer.
Référence : Ginet, M. (2003). Les clés de l’entretien avec le témoin ou la victime. Paris : La Documentation Française.
